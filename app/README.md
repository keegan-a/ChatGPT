# Phyllotaxis Drill Template

The React + TypeScript single-page app inside this directory renders phyllotaxis drilling templates sized for a 48" × 24" board.

## Available Scripts

- `npm run dev` – starts the development server with hot module replacement
- `npm run build` – type-checks the codebase and generates a production build in `dist/`
- `npm run preview` – serves the production build locally

## Key Directories

- `src/components/` – UI components, including the interactive preview canvas
- `src/utils/` – phyllotaxis math helpers and printer/export utilities
- `src/types/` – ambient TypeScript declarations

For repository-level instructions, see the root [`README.md`](../README.md).
